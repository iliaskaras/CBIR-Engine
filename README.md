CBIR, Content-Based image retrieval Project.<br />
Written in Java, a project that retrieve similar images based<br />
on their <b>colors</b> and <b>texture</b>.

Vectors used: <br />
•  <b> Histogram </b> <br />
• <b>  SURF </b> <br />

Distance Methods: <br />
•  <b> Divergence </b> <br />
•  <b> Euclidean </b> <br />
• <b>  Bhattacharyya </b> <br />
• <b>  Matusita </b> <br />


Written in Java and used PostgreSQL.
